# 便捷下载本地版

从零重写的 Android 本地版下载工具。项目不基于原加固 APK 重打包，不包含广告 SDK、登录系统或付费/订阅入口。

## v1.23 功能

- 首页入口：粘贴链接、支持列表、图片/视频/音频筛选、清空、粘贴、开始解析、下载教程。
- 平台解析：支持短链跳转、页面标题/描述、`og:image`、`og:video`、常见视频/图片/音频/m3u8 链接提取。
- 任意门嗅探：WebView 打开网页，拦截图片、视频、音频、m3u8/mpd 请求，并读取页面里的 `img/video/source/audio/a` 链接。
- 媒体选择：嗅探弹窗支持清空、图片过滤、全选、已选数量、媒体网格、下载选中内容。
- 媒体预览：视频/音频使用系统播放器，图片/GIF 使用可缩放预览，无法预览时可浏览器打开。
- 处理菜单：下载、复制链接、浏览器打开、重命名、更多；图片提供设为壁纸和编辑入口；视频提供提取音频、转 MP4、压缩、取帧入口。
- 下载任务：新增本地任务列表，展示缩略标签、类型、状态、进度和系统下载器提交结果；m3u8 先保存为“播放列表待处理”。
- 我的页：检查 GitHub Release 更新，有新版本时跳浏览器下载 APK。
- 悬浮窗：复制链接后可从悬浮窗快速进入解析页。

## 构建

```powershell
$env:JAVA_HOME='C:\Users\25021\Documents\app2\tools\jdk17\jdk-17.0.19+10'
$env:Path="$env:JAVA_HOME\bin;$env:Path"
.\gradlew.bat assembleDebug
```

APK 输出位置：

```text
app\build\outputs\apk\debug\app-debug.apk
```

## 说明

原 APK 使用加固壳，重打包后会闪退，因此本项目采用 clean-room 方式重新实现可见功能和本地工作流。涉及平台内容下载时，请只处理你有权保存或使用的内容。
