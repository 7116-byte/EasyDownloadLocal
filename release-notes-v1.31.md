# v1.31 任务管理、下载格式与悬浮球交互

- 任务页新增管理能力：全选、选择删除、清空任务、刷新。
- 下载文件名按媒体类型强制补后缀，避免视频/图片下载后没有正确格式。
- DownloadManager 增加 MIME 类型：视频、图片、GIF、音频、m3u8 分别设置对应类型。
- 长文件名截断时保留扩展名，避免 `.mp4/.jpg` 被截掉。
- 悬浮球交互改为：
  - 单击：解析剪贴板链接，并用小提示显示“解析到 1 个 MP4 / 3 个 JPG”等结果。
  - 双击：下载上次解析结果；如果还没解析，会先解析再下载。
  - 长按：打开软件并带入剪贴板链接。
- 工具页同步更新悬浮球使用说明。
