# v1.32 应用下载器与缓存保存

- 新增应用下载器：应用自动创建 `Download/EasyDownloadLocal`，下载结果保存到该目录。
- 下载前先检查应用媒体缓存；缩略图/预览已经缓存的资源会直接复制保存，未缓存才重新请求下载。
- 任务状态改为应用侧更新，显示“保存中 / 已保存 / 保存失败”和保存位置。
- 抖音多视频候选去重逻辑升级：按分辨率、文件大小、详情接口来源、无水印/下载地址特征选择最佳视频，不再只靠 URL 关键词。
- m3u8 资源会保存为播放列表链接文件，便于后续继续处理。

说明：如果平台短链生成的直链很快过期，建议先预览一次或立即下载；应用会尽量复用已加载缓存。
