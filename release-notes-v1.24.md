# v1.24 抖音解析与悬浮球补强

- 强化抖音解析：扫描转义后的页面 JSON，识别 `aweme_id/item_id/modal_id`，并尝试读取抖音详情接口。
- 强化媒体类型判断：无 `.mp4` 后缀的 `douyinvod`、`video/tos`、`play/playwm` 链接会识别为视频；`byteimg/imagex` 链接识别为图片。
- 修复预览打不开：视频、图片、GIF 预览请求带移动 UA 和 Referer，并保留浏览器打开兜底。
- 下载器请求增加 UA/Referer，减少 CDN 拒绝下载。
- 重做悬浮球：默认显示“粘”，检测到剪切板链接显示“待解析链接”；单击按下载范围解析下载，双击打开软件。
- 新增下载范围设置：视频、图片、音频、m3u8 可单独开关，主页解析和悬浮球下载共用。

说明：抖音网页接口可能因平台风控、登录态或地区限制返回不完整数据；本版会先走平台解析，失败时仍建议用任意门 WebView 嗅探获取素材。
