# v1.25 封面缩略图与抖音音视频分流

- 修复抖音解析结果里“视频只有音频”的问题：按 JSON 上下文字段区分 `video/play_addr/download_addr`、`music/play_url`、`cover/origin_cover/dynamic_cover`。
- 视频结果会绑定封面图，图片结果使用自身作为缩略图。
- 首页解析结果网格新增网络缩略图显示。
- 下载任务列表新增缩略图显示。
- 任意门嗅探弹窗新增缩略图显示，视频会使用已嗅探图片作为封面兜底。
- 下载和缩略图加载继续携带移动 UA 与 Referer，提高抖音/CDN 资源访问成功率。

说明：如果抖音接口因登录态或风控返回音频流和封面但不返回视频流，应用会显示可识别到的素材；这类链接建议进入任意门嗅探页面播放网页后再获取素材。
