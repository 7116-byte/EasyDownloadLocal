# v1.22 媒体识别与悬浮窗

- 解析结果新增媒体类型识别，可区分视频、图片、音频、m3u8/MPD 列表、网页和普通文件。
- 解析页面会提取 `og:video`、`og:image`、`twitter:image`、页面 `src/href` 等候选资源。
- 候选资源列表新增类型标签、来源、文件名、复制、打开和下载按钮。
- 新增悬浮窗服务，可在其他界面快速唤起解析器并读取剪贴板文本。
- 新增文本处理入口，系统选中文本后可分享到本地版解析。
- 新增“检查更新”按钮，会检查本项目 GitHub 最新 Release；发现新版本后显示下载按钮并跳浏览器下载 APK。
- 版本号更新到 `1.22`。
