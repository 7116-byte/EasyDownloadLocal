# v1.29 视频预览与重复视频修复

- 视频预览从系统 `VideoView` 升级为 Media3/ExoPlayer。
- 播放请求携带移动 UA、Referer，并允许跨协议重定向。
- 支持普通 MP4、m3u8/HLS、mpd/DASH 等更多视频流。
- 抖音同一作品解析出的多清晰度、多 CDN、多 bitrate 地址会归并成一个主视频。
- 主视频优先选择详情接口、download/play 地址和更高置信视频源，避免网格里重复 20 多个相同封面视频。

如果某条链接仍然黑屏，优先点“浏览器打开”验证平台是否已经限制直接播放；任意门播放后再获取素材通常能拿到真实播放请求。
